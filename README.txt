An open-source simple hack client for Pixel Worlds; MMO Sandbox.
Contains stuff like duping, AutoMath, AutoGM, and more!

This build uses MelonLoader v0.6.2 >> Download Link:
https://github.com/HerpDerpinstine/MelonLoader/releases/latest/download/MelonLoader.Installer.exe
